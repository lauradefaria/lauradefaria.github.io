Projects Highlights

Smart Refund

Final Project for .NET Course - Ada Tech & Mercado Eletrônico - 2024

Streamline the expense reimbursement process using advanced image processing techniques. This solution easily extracts crucial information from receipts and invoices, making expense detail entry quick, accurate, and efficient. It not only simplifies employee workflows but also speeds up financial department processes, allowing them to focus on strategic tasks. Simplify your workflow, eliminate errors, and save time.

Heat Map with Geotag Photos in João Pessoa

Digital Image Processing class project at UFPB - 2024

This project aims to identify the most visited places in João Pessoa using the Flickr platform to collect geotagged photos. These photos serve as the data source to generate a heat map highlighting the most frequented areas, and to segment it, allowing for more detailed analysis of regions of interest.

Capacitaded Vehicle Routing Problem with AI

AI class project at UFPB - 2023

The objective is to cluster instances from the literature and identify those similar to the studied article to gain knowledge about the set of high-quality instances available in the literature. It will assist in evaluating future algorithms that benchmark with these instances.

System for identifying salt and sugar in a solution

Project developed during the selection process for the LAIS laboratory at UFRN - 2020

Develop a system for the detection and analysis of salt and sugar in a solution, using the cyclic voltammetry technique. In order to accomplish this, concepts from electronics (operational amplifiers, current-to-voltage converter, filters, offset control) and programming (Python, Arduino, and ESP) were employed to simulate a prototype.

Low-cost computational platform for assessing aerobic conditioning in patients with spinal cord injuries

Research project at UFPB - 2019/2020

Development of a low-cost sensing device and the construction of control software focused on computational analysis of physical exercises adapted for patients with brain and spinal injuries.

Paraíba Humana e Inteligente

Software Develop at FAPESQ - 2020/2021

Foundation for Research Support in the State of Paraíba (FAPESQ) is a public institution, an integral part of the State Science and Technology System, with the aim of promoting the scientific and technological development of the State of Paraíba. Ammerican Kennel Club Group Classifier

AI class project at UFPB - 2023

The project aims to apply the knowledge of data processing and machine learning using a dataset containing information on 277 breeds extracted from the American Kennel Club website.
